# React E-Commerce App

A react project for e-commerce website


## Run Locally

Install dependcies

```bash
  npm intsall
```

To build, run the following command

```bash
  npm run build
```

To start live server, run the following command

```bash
  npm run start
```

## Home endpoint

```bash
  http://localhost:3000/
```


## Tech Stack

**Client:** HTML, CSS , bootstrap, react.js
